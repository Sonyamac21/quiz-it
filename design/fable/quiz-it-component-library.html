QUIZ-IT · COMPONENT LIBRARY

Every reusable component in every state. This file serves as both the Design System component reference and the state catalogue — hover and focus states are live (mouse over / tab to them). Components inherit all values from the Design Tokens file; nothing here introduces a new colour, size or curve. Host-only components are marked; the Display never renders dialogs, tables, spinners, links or form fields.
